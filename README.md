This repository covers the implementation of Bangla depressive post detection and analyses different embedding techniques with and without data augmentation.

This work has been publised to Annals of Emerging Technologies in Computing (AETiC). You can read the article from the following URL. https://aetic.theiaer.org/archive/v8/v8n3/p3.html

Authors: Saad Ahmed Sazan, ABM Rahman, Mahdi H. Miraz
